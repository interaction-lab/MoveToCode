Detailed Description

Class which supports components implementing Microsoft.MixedReality.Toolkit.Audio.IAudioInfluencer being used with audio sources.

AudioInfluencerController requires an AudioSource

component. If one is not attached, it will be added automatically. Each sound playing game object needs to have an AudioInfluencerController attached in order to have its audio influenced.
