protectedvirtual

Helper function to determine if the current profile is assigned to the active instance of MRTK.

In some cases profile data refers to other profile data in the MRTK config profile. In these cases, we don't want to render when the active instance isn't using this profile, because it may produce an inaccurate combination of settings.
