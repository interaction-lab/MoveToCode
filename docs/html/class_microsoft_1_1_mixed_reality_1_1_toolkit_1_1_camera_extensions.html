Returns true if a point is in the a cone inscribed into the Camera's frustum, false otherwise The cone is inscribed to a radius equal to the vertical height of the camera's FOV.

By default, the cone's tip is "chopped off" by an amount defined by the camera's far and near clip planes.

Parameters
pointPoint to test
coneAngleBufferDegreesDegrees to expand the cone radius by.
