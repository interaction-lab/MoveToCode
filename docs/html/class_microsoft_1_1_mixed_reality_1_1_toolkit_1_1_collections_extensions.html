static int Microsoft.MixedReality.Toolkit.CollectionsExtensions.SortedInsert< TElement > ( this List< TElement >  elements,
TElement  toInsert,
IComparer< TElement >  comparer = null 
)
static

Inserts an item in its sorted position into an already sorted collection.

This is useful if you need to consume the collection in between insertions and need it to stay correctly sorted the whole time. If you just need to insert a bunch of items and then consume the sorted collection at the end, it's faster to add all the elements and then use System.Collections.Generic.List<T>.Sort at the end.

Template Parameters
TElementThe type of element in the collection.
Parameters
elementsThe collection of sorted elements to be inserted into.
toInsertThe element to insert.
comparerThe comparer to use when sorting or null to use System.Collections.Generic.Comparer<T>.Default.
