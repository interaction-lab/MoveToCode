Detailed Description

Custom inspector for constraint manager.

Offers two modes depending on if auto constraint selection is active or not. In auto constraint selection mode, all constraints attached to the current game object will be displayed with respective goto buttons as well as an add button that allows adding new constraint components to the game object. In manual constraint selection mode, a list of user configured constraints is shown with options to modify the list, goto buttons as well as adding new constraints to the game object.
