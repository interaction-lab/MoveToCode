Detailed Description

ElasticsManager can be used to add elastics simulation to supporting components.

Call Initialize on manipulation start. Call ApplyHostTransform to apply elastics calculation to target transform. Elastics will continue simulating once manipulation ends through it's update function - to block the elastics auto update set EnableElasticsUpdate to false.
