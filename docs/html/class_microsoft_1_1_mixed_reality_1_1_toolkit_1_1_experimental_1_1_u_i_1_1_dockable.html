Detailed Description

Add a Dockable component to any object that has a Dockable and an Microsoft.MixedReality.Toolkit.UI.ObjectManipulator or Microsoft.MixedReality.Toolkit.UI.ManipulationHandler to make it dockable in Docks.

That allows this object to be used as part of a palette, shelf or navigation bar together with other objects.
