Detailed Description

This solver determines the position and orientation of an object as a directional indicator.

From the point of reference of the SolverHandler Tracked Target, this indicator will orient towards the DirectionalTarget supplied. If the Directional Target is deemed within view of our frame of reference, then all renderers under this Solver will be disabled. They will be enabled otherwise
