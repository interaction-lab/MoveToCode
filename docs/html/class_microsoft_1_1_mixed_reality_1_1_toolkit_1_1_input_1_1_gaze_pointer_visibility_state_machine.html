Detailed Description

Helper class for managing the visibility of the gaze pointer to match windows mixed reality and HoloLens 2 When application starts, gaze pointer is visible.

Then when articulate hands / motion controllers appear, hide the gaze cursor. Whenever user says "select", make the gaze cursor appear.

Has different behavior depending on whether or not eye gaze or head gaze in use - see comments on GazePointerState for more details.
