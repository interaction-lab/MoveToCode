Detailed Description

This class exists to route Microsoft.MixedReality.Toolkit.UI.PressableButton events through to Microsoft.MixedReality.Toolkit.UI.Interactable.

The result is being able to have physical touch call Interactable.OnPointerClicked. /summary>
