Detailed Description

The default implementation of the Microsoft.MixedReality.Toolkit.SceneSystem.IMixedRealitySceneSystem Because so much of this service's functionality is editor-only, it has been split into a partial class.

The default implementation of the Microsoft.MixedReality.Toolkit.SceneSystem.IMixedRealitySceneSystem This file handles the editor-oriented parts of the service.

This part handles the runtime parts of the service.
