Detailed Description

The BoundingBoxHelper class contains functions for getting geometric info from the non-axis-aligned bounding box of a GameObject.

These functions can be used to align another object to the center of a certain face or the center of an edge of a face... etc. The BoundingBoxHelper static function can be used for a one time calculation. The dynamic functions can be used to obtain boundingcube info on an object's Update loop. Operations are minimized in the dynamic use scenario.
