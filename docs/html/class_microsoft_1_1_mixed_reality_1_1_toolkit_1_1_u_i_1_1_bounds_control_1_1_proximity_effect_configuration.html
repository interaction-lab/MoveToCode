Detailed Description

Shareable configuration for a proximity effect that can be used with BoundsControl ProximityEffect scales and switches out materials for registered objects whenever a pointer is in proximity.

Scaling is done on three different stages: far / medium and close proximity whereas material switching will only be done on close proximity.
