Detailed Description

Utility class to control Microsoft.MixedReality.Toolkit.Input.PointerBehavior of pointers.

Hook up buttons to the public functions to turn rays on and off.

◆ TogglePointerEnabled< T >()

void Microsoft.MixedReality.Toolkit.UI.PointerBehaviorControls.TogglePointerEnabled< T > ( InputSourceType  inputType)

Toggles a pointer's "enabled" behavior.

If a pointer's Microsoft.MixedReality.Toolkit.Input.PointerBehavior is Default or AlwaysOn, set it to AlwaysOff. Otherwise, set the pointer's behavior to Default. Will set this state for all matching pointers.
