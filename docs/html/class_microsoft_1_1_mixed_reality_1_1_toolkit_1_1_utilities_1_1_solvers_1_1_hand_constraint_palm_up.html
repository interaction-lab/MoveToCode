Detailed Description

Augments the HandConstraint to also check if the palm is facing the user before activation.

This solver only works with Microsoft.MixedReality.Toolkit.Input.IMixedRealityHand controllers, with other Microsoft.MixedReality.Toolkit.Input.IMixedRealityController types this solver will behave just like its base class.

Determines if a controller meets the requirements for use with constraining the tracked object and determines if the palm is currently facing the user.

This function will modify the position and rotation behavior of the hand constraint if the followHandUntilFacingCamera variable is enabled.

Returns
