Detailed Description

Interface used by the focus provider to select the pointer that will be considered as primary.

The current primary pointer can we obtained via IMixedRealityFocusProvider.PrimaryPointer or subscribing to the primary pointer changed event via IMixedRealityFocusProvider.SubscribeToPrimaryPointerChanged.
