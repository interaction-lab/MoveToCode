bool Microsoft.MixedReality.Toolkit.SpatialAwareness.IMixedRealitySpatialAwarenessObserver.IsStationaryObserver
getset

Should the observer remain stationary in the scene?

Set IsStationaryObserver to false to move the volume with the user. If set to true, the origin will be 0,0,0 or the last known location.

◆ ObservationExtents

Vector3 Microsoft.MixedReality.Toolkit.SpatialAwareness.IMixedRealitySpatialAwarenessObserver.ObservationExtents
getset

Gets or sets the extents( 1/2 size) of the volume, in meters per axis, from which individual observations will be made.

When used when ObserverVolumeType is set to Microsoft.MixedReality.Toolkit.Utilities.VolumeType.Sphere the X value of the extents will be used as the radius.

◆ ObserverOrigin

Vector3 Microsoft.MixedReality.Toolkit.SpatialAwareness.IMixedRealitySpatialAwarenessObserver.ObserverOrigin
getset

Gets or sets the origin, in World Space, of the observer.

Moving the observer origin allows the spatial awareness system to locate and discard meshes as the user navigates the environment.
