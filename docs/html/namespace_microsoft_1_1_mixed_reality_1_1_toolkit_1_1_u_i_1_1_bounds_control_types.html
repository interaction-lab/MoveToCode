enum Microsoft.MixedReality.Toolkit.UI.BoundsControlTypes.WireframeType
strong

This enum describes which primitive type the wireframe portion of the bounds control consists of.

Wireframe refers to the thin linkage between the handles. When the handles are invisible the wireframe looks like an outline box around an object.
